# NodeDynamondbClientDemo

## Application version expectation:
```
npm: 6.7.0
node: 8.9.1 or above
```
## Setup Enviroment
Angular CLI:
```
npm install -g @angular/cli
```
## Run Project
### Open cmd and move to project folder
run:
```
  npm install
```
### Start Project
run: 
```
  npm start
```
```
  ### Your app will run on port 4200
  http://localhost:4200
```
